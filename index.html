BITCOIN MULTIPLE REGRESSION INFERENCE:

The aim of this analysis was to infer what variables affect bitcoin price the most, based on the size of coefficients of features which we included in our multiple regression model. The aim was certainly not to predict the bitcoin price, but to infer which features were massively associated with it, what kind of relationship they had with it, and whether this relationship was simple or more complex (interaction/polynomial terms).

Clustering Companies Using Cyber Security Metrics, Category Variables, and Text:

Part of my dissertation project. Real-time cyber security metrics data of different companies was provided by KYND(PVT.)LTD., which had to be wrangled and clustered using unsupervised machine learning techniques in R-Studio to find out what variables were leading to more or less cyber risks of any type.

Mapping Retail Outlets Based on Their Operational Status:

This is an analysis of Exceed Petroleum's database of all their retail assets in Pakistan. Retail sites were plotted on an interactive map using folium in python based on their file numbers and operational status. This was so that the management can easily run through and make their pricing plans considering how close a site was to its primary freight location.

See Project

Cooling Load Calculation Software Based on ASHRAE CLTD Method:

A load calculating software based on The American Society of Heating, Refrigerating and Air Conditioning Engineers (ASHRAE) standards. The Cooling Load Temperature Difference (CLTD) method was used to code the software in the form of a wizard. The wizard asks the user one by one about the specifics of the volume where the cooling load needs to be calculated.
